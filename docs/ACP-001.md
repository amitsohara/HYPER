# Principle
HyperMind possesses finite cognitive resources.

Attention is the mechanism that allocates these resources to maximize goal achievement while minimizing unnecessary computation.

Attention is dynamic.
Attention is explainable.
Attention is measurable.

Every allocation decision must be traceable.
