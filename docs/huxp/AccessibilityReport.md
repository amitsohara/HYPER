# Accessibility Report

## Keyboard Navigation
- All interactive elements (buttons, inputs, tabs) must be reachable via `Tab`.
- Logical DOM ordering.

## High Contrast
- Text contrast ratios exceed WCAG AA standards (e.g., `text-zinc-100` on `bg-zinc-950`).
- Status indicators use both color and icons (colorblind-safe).

## Screen Reader Support
- Semantic HTML tags (`<nav>`, `<main>`, `<aside>`).
- ARIA labels on icon-only buttons.
