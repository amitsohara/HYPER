# UI Style Guide

## Typography
- **Primary Font**: Inter (Sans-serif) for high legibility.
- **Mono Font**: JetBrains Mono for HCNS events, code blocks, logs, and IDs.

## Color Palette (Dark Mode First)
- **Base**: `zinc-950`
- **Surface**: `zinc-900`, `zinc-800`
- **Text**: `zinc-100` (primary), `zinc-400` (secondary)
- **Status Colors**:
  - Healthy/Active: `emerald-400`
  - Warning/Pending: `amber-400`
  - Error/Failed: `red-400`
  - Cognitive/Thinking: `indigo-400`

## Components
- **Cards**: Rounded (`rounded-xl`), border (`border-zinc-800`), flat shadows.
- **Buttons**: Minimalist padding, hover transition, clear active states.
- **Badges**: Pill-shaped, semi-transparent background (`bg-emerald-400/10`).
