# Navigation Architecture

## Principles
- **Mission-Centric Navigation**: Everything revolves around missions.
- **Progressive Disclosure**: Advanced cognitive views are available only when requested.
- **Single Source of Truth**: No duplicate navigation paths or redundant dashboards.

## Top-Level Navigation
1. **Home**: Launch pad, recent missions, quick actions.
2. **Mission Queue**: Central screen for all active, pending, and completed missions.
3. **Mission Control**: The runtime interface for a selected mission.
4. **Cognitive Observatory**: Aggregated analytics, telemetry, and system health.
5. **Knowledge Center**: Unified memory and skills repository.
6. **Settings**: Configuration and preferences.
