# Evidence Model

## Sources
- **Perception:** Raw sensor data, transcripts, object detection.
- **World Model:** Spatial/temporal state of entities.
- **Reasoning:** Premises and conclusions generated by HDME.
- **Simulation:** Metrics and narrative outcomes from HSTE.
- **Execution:** Logs of actions taken and their success/failure.

## Confidence
- Each piece of evidence carries a confidence score (0.0 - 1.0).
- Final mission confidence is an aggregate of individual evidence scores.
