# Mission Score Specification

## Components
- **Success:** Did it achieve the directive? (40%)
- **Efficiency:** Time taken vs expected. (20%)
- **Safety:** Were boundary conditions met? (20%)
- **Knowledge Gain:** Were new concepts learned? (20%)

## Calculation
`MissionScoringEngine` computes the weighted average of these components to produce the overall mission score (0 - 100%).
